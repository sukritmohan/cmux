06 / Minimap

Live Workspace Overview

Pinch out on any view (or tap "Overview" in the pane type dropdown) to reveal a bird's-eye view of the entire desktop workspace. Every pane renders live content in real time — this is a full progress tracker, not a static snapshot. You see everything happening across your project at once: builds running, logs streaming, servers responding, agents working.

Trigger
Pinch Out + Dropdown
Pinch out from any view zooms to minimap. Also accessible via "Overview" in the pane type dropdown — always two taps away. Pinch in or tap a pane to zoom back.
Live Content
Real-Time Pane Previews
Every pane card shows live content from the desktop — terminal output streams, browser page renders, file lists update. PTY subscriptions remain active. A pulsing "LIVE" indicator confirms real-time streaming.
Multi-Surface Panes
Stacked Cards with Fan-Out
When a desktop pane has multiple surfaces (e.g., 3 terminal tabs), the minimap shows stacked cards with a count badge. Tap the stack to fan out cards inline — then tap a specific card to zoom into that surface.
Mobile ↔ Desktop
Flat Tab Strip vs. Spatial Layout
Mobile flattens all surfaces per type into one scrollable tab strip (Terminal shows all terminals, Browser shows all tabs). The minimap is the only place on mobile where you see the actual desktop spatial arrangement — pane columns, proportions, and stacking.

09 / Interactions

Gesture Map

Every interaction in the pane type switcher and surrounding navigation.

Gesture Action Feedback
Tap pane type icon Open/close dropdown Scrim dims content, dropdown scales in from top-right origin
Tap dropdown item Switch pane type Dropdown closes, top bar cross-fades to new type's tabs
Tap scrim Close dropdown Dropdown scales out, scrim fades
Tap tab Switch surface Active underline slides to new tab, content cross-fades
Left edge swipe Open workspace drawer Frosted panel slides in with spring easing
Tap (+) fan Expand modifier keys Esc, Ctrl, Alt, Cmd, Tab fan out above button
Tap clipboard fan Expand clipboard actions Copy, Paste, Cut fan out above button
Long press (+) Customize quick keys Settings sheet slides up
Pinch out Minimap view Terminal zooms out to show all panes at desktop proportions
