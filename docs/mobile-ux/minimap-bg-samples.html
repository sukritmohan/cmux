Minimap Background — Visual Samples

Three options for the minimap background. Each shown in dark + light mode with the same workspace layout: 2 terminal panes (one with stacked surfaces), 1 browser pane. Pick the one that feels right.

Option A — Deep Blur + Dim

The workspace fades into a frosted, color-tinted haze. Subtle radial glows from terminal green and browser blue bleed through. Pane cards float above with crisp edges. Immersive — you feel like you're "pulling back" from the workspace.
