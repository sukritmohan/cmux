cmux Mobile Companion

Pane Type Switcher

A segmented control replaces the dropdown menu. All four pane types are visible at a glance -- one tap to switch, zero cognitive overhead. Frosted glass surfaces, floating toolbars, and spring-physics transitions make every interaction feel physical and alive.

Interaction Model

The segmented control makes all pane types visible at once. No dropdown, no hidden menu. The sliding pill provides instant spatial feedback -- you always know where you are and where you can go.

Segmented Control

A frosted-glass pill at the top with 4 segments. The active segment has a sliding white pill background that animates with spring physics (tension 0.2, friction 0.8). Icons in each segment; the active one optionally reveals a text label. One tap to switch -- no intermediary menu.

Tab Strip Adapts

Below the segmented control, a context-sensitive tab strip appears. Terminal and Browser show scrollable pill-shaped tabs. File Explorer shows breadcrumb pills. Shell shows session tabs. The strip cross-fades when switching pane types.

Floating Toolbar

The bottom bar is a floating frosted-glass pill, not a hard-edged bar. It hovers above the bottom edge with rounded corners, backdrop blur, and soft shadow. Each pane type swaps in its own controls with a subtle spring animation.

Transition Animations

Every transition uses spring physics. No linear or ease-in-out curves. The goal is a physical, haptic-feel interface where elements have mass and momentum.

Segment Switch

Sliding pill animates with spring curve (500ms cubic-bezier(0.2, 0.8, 0.2, 1)). The pill width stretches ~12% at the midpoint of travel, creating an elastic, physical feel. Content area cross-fades: outgoing fades to 0 + translates 12px in the swipe direction (150ms), incoming fades from 0 + translates from 12px opposite (200ms, 50ms delay). Tab strip cross-fades in place (150ms). Floating toolbar slides up 4px and cross-fades (200ms). Total perceived transition: under 350ms.

Workspace Drawer Open

Drawer slides in from left with spring curve (400ms). Background content scales to 0.95 and dims. Scrim fades in with backdrop blur (200ms). Drawer items stagger with 40ms delay each. Close: drawer slides out (300ms, faster for responsive feel), content scales back to 1.0.

Haptic Feedback Points

Light impact on segment tap (before animation starts). Medium impact when sliding pill reaches destination. Light impact on workspace card tap. Selection feedback on tab pill tap. All haptics pair with the visual moment -- never before, never after.
